# ☕ Cozy Coffee 网页项目

一个使用 **React** 构建的练习型咖啡店网站，包含菜单、介绍、联系方式等页面。该项目旨在练习前端页面结构设计、组件拆分、样式管理与 React Router 应用。

---

## ✨ 项目特点

- 多页面导航（使用 React Router）
- 模糊背景图 + 页面内容区域背景色
- 菜单页面展示卡片式饮品项
- 样式按页面分离，便于维护
- 使用 Lucide 图标库提升视觉表现

---

## 🛠 技术栈

- [React](https://reactjs.org/)
- [Vite](https://vitejs.dev/)
- [React Router DOM](https://reactrouter.com/)
- [Lucide-react](https://lucide.dev/)
- 原生 CSS
- 图片资源：本地 + Unsplash 外链
